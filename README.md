# codealpha_task_3_
## 📌 Overview
This is a simple rule-based chatbot built using Python.
The chatbot interacts with users through the console and responds based on predefined conditions.
It is designed for beginners who want to understand the fundamentals of chatbot development.
-

## 🎯 Features
Responds to greetings like "hello", "hi"
Answers simple predefined questions
Uses if-else logic for responses
Runs in a continuous loop until user exits
Simple and beginner-friendly code
- 

## 🛠️ Technologies Used
Python 3
-

## Basic concepts: if-else statements , while loop , String methods , User input/output
